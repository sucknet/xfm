# XFlames Token

XFlames (XFLM) - Shine a light on X1, powering Flameriver and Flamebeam

## Features

- Token Information Display
- Contract Address with Copy Function
- Supply: 10,000 XFLM
- Modern, Minimalist UI

## Deployment

This project is deployed on Railway.

## Local Development

```bash
npm install
npm run server
```

Open http://localhost:3000
